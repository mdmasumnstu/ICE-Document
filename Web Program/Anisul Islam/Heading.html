Singly Linked List

A singly linked list is a fundamental data structure in computer science and programming. It is a collection of nodes where each node contains a data field and a reference (link) to the next node in the sequence.

The last node in the list points to null, indicating the end of the list. The data part stores the actual information, while the pointer (or reference) part stores the address of the next node in the sequence.
